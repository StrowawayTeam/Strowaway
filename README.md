# Strowaway
Strowaway Antivirus — антивирус для Windows с модульной архитектурой. Не требует установки Python. Включает сканер файлов, карантин, мониторинг процессов и сети, защиту системных файлов, онлайн-проверку через VirusTotal, планировщик, резервное копирование и контроль веб-камеры/микрофона.
